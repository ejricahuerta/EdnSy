# Local Citations Checklist (Toronto and Ontario)

This is an execution checklist for off-page local SEO work that should be done outside the codebase.

## Priority 1: Core listings

- [ ] Claim and fully optimize **Google Business Profile**
  - [ ] Primary category: AI consultant / Automation service (pick closest fit)
  - [ ] Add service descriptions (Voice AI, Business Automation, Website and SEO)
  - [ ] Add service areas (Toronto, GTA, Ontario)
  - [ ] Add business photos, logo, and consistent branding
  - [ ] Add booking and website links
- [ ] Create/optimize **Bing Places**
- [ ] Create/optimize **Apple Business Connect**

## Priority 2: Trusted directories

- [ ] Yelp
- [ ] YellowPages Canada
- [ ] Better Business Bureau profile (if applicable)
- [ ] Clutch (for B2B services)
- [ ] Cylex / profile directories relevant to local services

## Priority 3: Data consistency

- [ ] Keep NAP fields consistent everywhere:
  - [ ] Business name
  - [ ] Address/service area format
  - [ ] Contact email
  - [ ] Website URL (`https://ednsy.com`)
- [ ] Use the same service wording across listings
- [ ] Add UTM parameters to listing links for attribution tracking

## Reviews and trust signals

- [ ] Request Google reviews from happy clients
- [ ] Reply to every review (positive and negative)
- [ ] Add recent project updates/posts in GBP monthly
- [ ] Publish at least one photo/video update monthly

## Quarterly maintenance

- [ ] Audit and fix duplicate listings
- [ ] Update hours, services, and descriptions
- [ ] Check for broken website links in all directories
- [ ] Export listing performance metrics and compare quarter over quarter

# Lead Rosetta — Taskmaster tasks

Tasks for the marketing app are in `tasks.json`. They are aligned with [apps/marketing/docs/prd.md](../docs/prd.md).

## Using Taskmaster

- **CLI:** From repo root: `task-master list -f apps/marketing/tasks/tasks.json`
- **MCP:** Use `file`: `apps/marketing/tasks/tasks.json` and `projectRoot`: repo root when calling get_tasks, next_task, set_task_status, etc.

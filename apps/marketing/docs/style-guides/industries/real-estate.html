05 ‚Äî Usage Notes for Lead Rosetta
Dynamic fields to populate
Replace [agent_name], [agency_name], [initials], [phone], [years_exp], [sold_volume], [homes_sold]. Agent card initials auto-generated from name field.
Stat defaults (if CSV missing)
Use $50M+ sold, 100+ homes, 97% list price, 10+ years GTA. These are conservative enough to be true for almost any established Toronto agent without fabricating specific numbers.
Property listings
Show 3 placeholder listing cards ‚Äî always include one "Sold" to signal track record. If the agent has real listings in the CSV, pull addresses. Otherwise use realistic Toronto neighbourhood placeholders like Rosedale, Danforth, Forest Hill.
Contact card in hero
The inline contact form on the hero right is the conversion mechanism. Never remove it ‚Äî for real estate, the first contact is everything. Keep the three fields: name, contact, and intent (buying/selling/both).
Dark hero rationale
Obsidian dark with champagne accents signals prestige and confidence. Toronto real estate buyers respond to this aesthetic ‚Äî it communicates premium market positioning. Never use bright colours or gradients in this template.
Licence / credential rule
Always include RECO Licensed and TRREB Member in the agent card and footer. These are trust signals every Ontario buyer looks for. If not in CSV, include both as defaults ‚Äî all licensed Ontario agents have them.

# Apify

Use your Apify API key to run Google Maps imports directly from Prospects.

In this shared workspace, the saved key is workspace-wide (all users).

## 1) Get your API key

1. Sign in to your Apify account.
2. Go to **Settings** and copy your API token.
3. In this app, open **Dashboard → Integrations → Apify**.
4. Paste the key and click **Save API key**.

## 2) Start an import

1. Open **Dashboard → Prospects**.
2. Click **Import from Apify**.
3. Choose an industry and location, then start the import.

The import runs in the background. New prospects appear when the job completes.

## Notes

- You can update or remove the key anytime from Integrations.
- The key is stored securely in server-side settings and shared across all workspace users.

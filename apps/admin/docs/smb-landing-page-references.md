## How to Use in Your Prompt

Always specify **Next.js** (App Router) and **Tailwind CSS** so v0 generates compatible code.

Single reference:
```
Build a landing page with Next.js (App Router) and Tailwind CSS.
Reference: sweetgreen.com vibes — light, clean, lots of white space,
fresh green accents, editorial food photography
```

Mix two references:
```
Build a landing page with Next.js (App Router) and Tailwind CSS.
Reference: equinox.com layout with allbirds.com color warmth
```

Full example (industry + stack):
```
Create an SMB landing page using Next.js (App Router) and Tailwind CSS.
Reference: tendental.com — modern dental, warm and inviting, booking widget above the fold.
Include: hero, services grid, insurance logos, and location finder.
```
